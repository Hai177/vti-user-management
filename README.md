# Run this project and follow commands

## Require

- NodeJs
- Postgresql

## In packages

Create .env file. Then copy the environment from env.example file. Change the environment variable to match your config

## Install packages

```bash
npm install
```

or

```bash
yarn
```

## Seed database

```bash
npm run seed
```

or

```bash
yarn seed
```

## Run source code at localhost

```bash
npm run dev
```

or

```bash
yarn dev
```

## Format

```bash
npm run format
```

or

```bash
yarn format
```
